# 🎯 ReviewQR Pro

> A free, beautiful, and customizable QR code poster generator for Google Business reviews

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://your-username.github.io/your-repo-name/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## ✨ Features

- 🎨 **Customizable Design** - Three beautiful templates (Google, Modern, Elegant)
- 🌍 **Multi-language Support** - English, Spanish, Portuguese, Hindi, Urdu, Arabic
- 🎨 **Color Customization** - Adjust colors for background, text, QR code, and accents
- 🖼️ **Logo Upload** - Add your business logo to QR code center
- 📱 **Responsive Preview** - Real-time preview of your poster design
- 💾 **Multiple Export Options** - Download as PNG or PDF
- 🖨️ **Print Ready** - High-resolution output optimized for printing
- 📤 **Social Sharing** - Easy sharing on Facebook, Twitter, LinkedIn
- ⚡ **Fast & Client-Side** - No backend required, all processing done in browser

- **React 19** - UI framework

## 📦 Installation

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   cd your-repo-name
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000`

## 🎯 Usage

1. **Enter Your Google Review URL**
   - Get your Google Business review link from your Google Business Profile
   - Paste it in the "Google Review URL" field

2. **Customize Your Poster**
   - Enter your business name
   - Choose a language
   - Select a template style (Google, Modern, or Elegant)
   - Adjust colors to match your brand
   - Upload your business logo (optional)

3. **Export Your Poster**
   - Click "Download PNG Image" for a high-resolution image
   - Click "Print / Save as PDF" to generate a printable PDF

4. **Share the Tool**
   - Use the social sharing buttons to share this free tool

## 🏗️ Build for Production

```bash
npm run build
```

The build output will be in the `dist/` directory.
